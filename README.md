# rustdl

## What

Download multiple youtube videos at once.

## Running

```
cargo run main.rs
```


## Dependencies


```
sudo apt-get install -y \
  libwebkit2gtk-4.0-dev \
  libjavascriptcoregtk-4.0-dev \
  build-essential \
  libgtk-3-dev \
  libpango1.0-dev \
  librust-atk-dev \
  libgdk-pixbuf-2.0-dev \
  libcairo2-dev \
  libjpeg-dev \
  libgif-dev \
  libsoup2.4-dev \
  libdbus-1-dev
```
